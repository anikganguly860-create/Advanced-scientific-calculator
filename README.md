# Ultra Advanced Scientific Calculator

A modern, mobile-friendly scientific calculator built with HTML, CSS, and JavaScript. Features include:

- Full scientific functions: `sin`, `cos`, `tan`, `sec`, `cosec`, `cot`, `log`, `ln`, `√`, factorial
- Graph plotting with `x`, zoom, and pan (optional open/close)
- Live digital clock
- Dark & Light theme toggle
- Animated buttons for smooth UI
- Built-in multi-currency converter with major world currencies
- Fast and ultra-smooth performance, optimized for mobile and desktop

## How to Use

1. Open `index.html` in a browser or in Acode.
2. Use buttons for calculations or graphing.
3. Toggle theme or currency mode with buttons.
4. Graph functions using variable `x`.
